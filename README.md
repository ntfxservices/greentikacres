# Greentik Acres — Premium Land Investment Platform

A premium, transparent, and mobile-first verified land investment landing page built using Next.js, React, and Tailwind CSS.

## Features

- **Redesigned Premium UI**: High-contrast, elegant colors, and customized typography.
- **Responsive Navigation**: Transitions seamlessly on scroll and includes a mobile-first side overlay.
- **Stats Display**: Interactive animated counters displaying key company trust indicators.
- **Land Pooling details**: Custom pooling features detailing managed resource combination, benefits, and how it works.
- **Complete Verification Focus**: Highlighted trust pillars representing title check, local registration support, and legal clearances.

## Technology Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v4, Vanilla CSS Animations
- **Icons**: Lucide React
- **Typography**: Inter (Google Fonts)

## Setup and Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser to view the page.
